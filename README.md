# ❄️ Winter Arc: ML Engineering

> No days off. One algorithm at a time.

A structured, day-by-day machine learning challenge — built from scratch as a personal commitment to mastering ML fundamentals before stepping into industry. Each day is a new concept, a new notebook, zero excuses.

---

## 🧩 Problem Statement

Most ML learners jump between random tutorials and end up with shallow, scattered knowledge. This repo is the antidote — a disciplined, sequential grind through core ML algorithms, applied hands-on with real datasets in Jupyter, every single day.

## 🛠️ Tech Stack

- **Language:** Python 3.x
- **Environment:** Jupyter Notebook / Dev Container
- **Core Libraries:** `scikit-learn`, `xgboost`, `pandas`, `numpy`, `matplotlib`, `seaborn`

---

## 🚀 How to Run

**1. Clone the repo**
```bash
git clone https://github.com/bynaif/Winter_Arc_ML.git
cd Winter_Arc_ML
```

**2. Install dependencies**
```bash
pip install scikit-learn xgboost pandas numpy matplotlib seaborn notebook
```

**3. Launch Jupyter**
```bash
jupyter notebook
```

**4. Navigate to any Day folder and open the `.ipynb` file.**

> Alternatively, open the repo in VS Code with the Dev Container config included — zero setup required.
